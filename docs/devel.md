# Application menu:

1. Template:
   1. Git repo
   2. Author
   3. Template name
2. Layout:
   1. Page size (A4, letter, ...)
   2. Page margins (left, right, top, bottom)
      1. default (2.5mm)
   3. Page orientation (portrait, landscape)
   4. Units (mm, px, in, cm)
      1. default (mm)
   5. DPI
      1. default (300)
3. Sections
   1. Header / Footer / Content
   2. Header height
   3. Footer height
   4. Automatic page numbering
